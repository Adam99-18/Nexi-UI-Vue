## 五、关键设计决策

| 决策 | 理由 |
|------|------|
| FunctionalComponent + h() 渲染 Action Popconfirm/Tooltip | 避免 v-bind 污染 a-button props |
| loadingMap ref 在 computed 之前声明 | 解决 TDZ |
| { ...item } 不可变 spread | 防止修改原始 props |
| getContainer 统一 `\|\| document.body` | SSR 兼容 + 容器不存在时降级 |
| a-switch @change 替代外层 div @click | 消除事件冲突 |

## 六、接入指南

```js
// main.js — 最小配置
import { createApp } from 'vue'
import Antd from 'ant-design-vue'
import 'ant-design-vue/dist/reset.css'
import NexiUI from 'nexi-ui'
import 'nexi-ui/style.css'
import App from './App.vue'

const app = createApp(App)
app.use(Antd)     // 必须先注册
app.use(NexiUI)
app.mount('#app')
```
